# Changelog

## v0.1.0 - 2026-06-23

Initial public release.

### Added

- Reusable Codex skill for syncing webpages into Feishu/Lark Docs and Wiki.
- Page-type rules for technical docs, image-rich articles, and visual landing pages.
- Feishu/Lark write and verification workflow.
- Markdown assembly helper for structured extraction JSON.
- Public example JSON for deterministic Markdown generation.

### Boundaries

- Does not inspect cookies, local storage, saved passwords, or session stores.
- Does not bypass CAPTCHA, MFA, paywalls, or missing permissions.
- Does not attempt pixel-perfect landing page reconstruction in Feishu.
